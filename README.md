# VoidScope

A cinematic movie & TV discovery app built with Next.js 15, TypeScript, and TanStack Query.

## Features
- Real-time movie search
- Advanced filtering
- Persistent watchlist using Zustand
- Nebula glass dark design system

## Tech Stack
- Next.js 15
- TypeScript
- TanStack Query
- Zustand
- TailwindCSS

## Run locally

npm install
npm run dev
